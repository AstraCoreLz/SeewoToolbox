## 简介

希沃工具箱是由 **我是从前那个少年** 开发的免费工具集，专为学校希沃交互平板及相关 Windows 设备设计。

日常教学和维护中，老师们经常需要反复安装软件、冻结/解冻磁盘、激活系统等操作，本工具将这些常用功能整合到一个简洁美观的界面中，**一键完成，告别繁琐**。

---

## 功能一览

| 分类 | 功能 |
|------|------|
| **常用软件** | 安装微信（32/64 位可选）、安装 QQ、安装腾讯电脑管家 |
| **希沃工具** | 一键开荒（9 款希沃教学软件自选安装）、解冻/冻结磁盘 |
| **系统工具** | 激活 Windows、重启到恢复模式、下载原版 Windows 镜像 |

### 一键开荒包含软件

希沃白板5、希沃视频展台、班级授课助手、希沃管家、轻录播、PPT小助手、WPS Office、Office 2016 四合一带激活、希沃桌面壁纸
